URL shortener microservice

How to use?

Call /api/shorturl with a POST method and the url you want to shorten as the "url" parameter

The API will respond with JSON including the original url and the new one, example: {"original_url": https://www.youtube.com, "short_url": 4}

Calling /api/shorturl/[short_url] will then redirect you directly to your desired url!
